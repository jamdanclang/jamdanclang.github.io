Adjunct Instructor - Doane University

I've developed and instructed graduate courses in Doane University's Instructional Design and Open Learning programs.

The Masters of Science in Instructional Design and Technology online degree program prepares individuals to lead in the design, development, and integration of technology and online learning into academic and private sectors. The program focuses on the instructional design process, adult learning theory, communication, media, and technology.

Game Strategies and Motivation

This course explores why we are so motivated by games and how to use game mechanics to capture this motivation in non-game contexts. Gamification is used in more places than one would expect. Understanding these dynamics allows the effective and enjoyable implementation of gamified systems.

COURSE OUTCOMES
After completing these courses, students will be able to:

  1. Differentiate between games and gamified systems
  2. Identify dynamics and mechanics of gamified systems and how they are used to motivate players
  3. Distinguish gaming models and methods as they relate to academic theories and psychological methods
  4. Recognize the ethical implications of gamification
  5. Generate a solution to a problem using gamification strategies

Advanced Web Design

The web needs to be user-friendly and accessible. This course covers advanced web design concepts including UI/UX, Accessibility Standards, Responsive Web Design, and Optimization of web content.

COURSE OUTCOMES
After completing this course, students will be able to:

  1. Utilize HTML, CSS, and JS to create interactive user experiences
  2. Execute key principles of web design and User Experience/User Interface
  3. Recognize the need for accessibility and inclusivity on the web
  4. Design web content for multiple viewing environments
  5. Interpret code from external sources
  6. Construct a multi-page website for a specific educational purpose and audience
